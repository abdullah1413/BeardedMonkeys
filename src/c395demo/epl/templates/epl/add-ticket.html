Create Tickets Page

Hardware Tickets:

For if the problem is with a device (for example, a computer not turning on or a tablet's screen being unresponsive).

+

Software Tickets:

For if the problem is with an application, program, or system (for example, a program causing your computer to crash, or an app displaying improper data).

+

Service Tickets:

For if you want to request a certain service such as moving equipment, gaining access to a new system, or obtaining new equipment or software.

+

Password Tickets:

For if you need to reset your password for one of the systems of the Edmonton Public Library.

+

General/Other Tickets:

For if your problem doesn't seem to fall into any of the other categories.
